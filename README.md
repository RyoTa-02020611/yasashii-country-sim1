# やさしい国家運営ゲーム

〜ノヴァリア王国を救う、経済×政策シミュレーション〜

## 📖 概要

「やさしい国家運営ゲーム」は、高校生でも遊べる国家運営シミュレーションゲームです。プレイヤーはノヴァリア王国の統治者となり、6人のアドバイザーの意見を参考にしながら、政策を選択して国を運営していきます。

物価・失業率・生活しやすさ・国庫残高の4つの指標をバランスよく管理することが重要です。経済政策や社会政策の影響を体験しながら、国家運営の難しさと面白さを学ぶことができます。

教育現場で使える教材として設計されており、教師向けのダッシュボード機能も備えています。

## 🌍 世界観・ストーリー

### ノヴァリア王国

ノヴァリア王国は、光の石という特殊な資源を有する小さな王国です。かつては豊かな国でしたが、最近になって様々な問題が発生し始めています。

### 光の石

光の石は、ノヴァリア王国の重要な資源です。この石の価格変動が、国の経済に大きな影響を与えます。光の石の価格が急騰すると、生活必需品の価格も上がり、国民の生活が圧迫されます。

### 王家の危機

物価の上昇、失業率の増加、財政の悪化、生活の質の低下...。ノヴァリア王国は危機に直面しています。プレイヤーは統治者として、これらの問題に立ち向かい、国を立て直す必要があります。

6人のアドバイザーがそれぞれの専門分野から意見を述べますが、最終的な判断はプレイヤーに委ねられます。正しい政策選択が、ノヴァリア王国の未来を決めるのです。

## 🎮 ゲームの遊び方（初心者向け）

### 1. イベントを読む

各ターンで、ノヴァリア王国で発生するイベントが表示されます。例えば「パンの値段が急に3倍に！」や「大規模な自然災害が発生」など、様々な出来事が起こります。イベントをよく読んで、どのような問題が発生しているかを理解しましょう。

### 2. 政策を選ぶ

イベントを読んだら、次は政策カードから1つを選択します。6人のアドバイザーがそれぞれの視点から意見を述べるので、それを参考にしながら最適な政策を選びましょう。

- **リク（財務）**: 財政と税制の専門家
- **ハル（経済）**: 経済政策の専門家
- **サト（社会）**: 社会政策の専門家
- **ツムギ（外交）**: 外交と貿易の専門家
- **ミナ（市民）**: 市民の声を代弁
- **ナビ（データ分析）**: データに基づく分析を提供

### 3. CFO行動フェーズ

政策を選んだ後は、CFO（財務責任者）として追加の行動を1つ選べます。以下の4つから選択できます：

- **ETF取引**: 光の石ETFを売買して、短期的に財政を改善します
- **外交交渉**: 他国と交渉して、資源価格の安定化を図ります
- **市民調査**: 市民の声を聞いて、生活しやすさを向上させます
- **産業育成**: 特定産業に投資して、長期的な効果を得ます

### 4. メーターを管理

ゲーム中は、4つのメーターを常に確認しましょう：

- **💰 物価**: 生活必需品の価格レベル。高すぎると国民の生活が圧迫されます
- **👥 失業率**: 労働人口に占める失業者の割合。高すぎると社会不安が広がります
- **🏠 生活しやすさ**: 国民の生活の質。物価、雇用、社会保障などが影響します
- **💎 国庫残高**: 国の財政状況。マイナスになると借金が増え、財政破綻のリスクがあります

これらのメーターが危険な値になると、ゲームオーバーになる可能性があります。

### 5. 振り返りレポートを見る

各ターン終了後、振り返りレポートが表示されます。そのターンでどのような政策を選んだか、どのアドバイザーの意見を参考にしたか、メーターがどのように変化したかがまとめられます。自分の判断を振り返ることで、次回のプレイに活かせます。

### 6. 結果画面で評価される

章をクリアするか、失敗条件に達すると、結果画面が表示されます。プレイしたターン数、最も参考にしたアドバイザー、最も改善した指標などが表示され、自分のプレイスタイルを確認できます。

## 👨‍🏫 授業向けの説明（教師向け）

### 教師用ダッシュボード

ゲームには「教師モード」があり、プレイヤーの判断傾向を分析するダッシュボードが用意されています。このダッシュボードでは、以下の情報を確認できます：

- 最も多く採用したアドバイザー
- 各アドバイザーの採用回数
- 各メーターの改善量・悪化量
- 典型的なミスパターンと授業で触れるべきポイント

### 判断傾向の把握

ダッシュボードでは、プレイヤーがどのような判断をしたかを可視化します。例えば、「財政重視の政策を多く選んだ」「失業対策を優先した」など、プレイヤーの傾向が一目で分かります。

### シナリオのテーマ別学習内容

ゲームは6つの章で構成されており、それぞれ異なるテーマを扱います：

#### 第1章：国の危機
- **学習内容**: ゲームの基本に慣れる入門シナリオ
- **重点**: 4つのメーターの意味と、政策選択の基本を理解する

#### 第2章：物価の乱れ
- **学習内容**: インフレと物価の管理
- **重点**: インフレが国民生活に与える影響、物価安定化の政策手段
- **教材として使えるポイント**:
  - インフレの原因（需要と供給、貨幣供給量など）
  - 物価安定化のための政策（金融政策、財政政策）
  - インフレとデフレの違い

#### 第3章：仕事がなくなる
- **学習内容**: 失業率の上昇と景気の悪化
- **重点**: 雇用創出の重要性、失業対策の政策手段
- **教材として使えるポイント**:
  - 失業の種類（構造的失業、循環的失業など）
  - 雇用創出のための政策（公共事業、職業訓練など）
  - 失業率と景気の関係

#### 第4章：借金の増加
- **学習内容**: 財政の悪化を食い止め、節約と増税のバランスを取る
- **重点**: 財政健全化の重要性、税収と歳出のバランス
- **教材として使えるポイント**:
  - 財政赤字と借金の関係
  - 財政再建のための政策（歳出削減、増税など）
  - 財政政策の限界

#### 第5章：外交トラブル
- **学習内容**: 資源ショックと国際価格の変動
- **重点**: 外交と資源管理の重要性、国際経済の影響
- **教材として使えるポイント**:
  - 貿易と国際経済
  - 資源依存のリスク
  - 外交政策と経済政策の関係

#### 最終章：総決算
- **学習内容**: これまでの学びを総動員する総合演習
- **重点**: すべての指標をバランスよく管理する
- **教材として使えるポイント**:
  - 複数の政策目標のトレードオフ
  - 総合的な政策判断
  - 国家運営の複雑さ

### 教材として使えるポイント

- **体験型学習**: 実際に政策を選択することで、理論だけでなく実践的な理解が深まります
- **データ分析**: 教師ダッシュボードで、プレイヤーの判断傾向を可視化できます
- **振り返り**: 各ターンの振り返りレポートで、自分の判断を客観視できます
- **段階的学習**: 章ごとにテーマが分かれているため、授業の進度に合わせて使用できます

## ✨ 機能一覧

### コア機能

- ✅ **メーターUI**: 4つのメーター（物価・失業率・生活しやすさ・国庫残高）を視覚的に表示

- ✅ **アドバイザー会話**: 6人のアドバイザーがそれぞれの専門分野から意見を述べる
- ✅ **政策カード**: 複数の政策から1つを選択し、メーターに影響を与える

### CFO行動フェーズ

- ✅ **ETF取引**: 光の石ETFを売買して、短期的に財政を改善
- ✅ **外交交渉**: 他国と交渉して、資源価格の安定化を図る（成功率あり）

### UI/UX

- ✅ **レスポンシブデザイン**: PCとモバイルの両方に対応
- ✅ **統一されたテーマ**: ダークテーマ（slateベース）で統一されたUI

## 🚀 セットアップ方法

### 必要な環境

- Node.js 18以上
- npm または yarn

### インストール手順

1. **リポジトリをクローン**
   ```bash
   git clone https://github.com/your-username/nova-nation-game.git
   cd nova-nation-game
   ```

2. **依存関係をインストール**
   ```bash
   npm install
   ```

3. **開発サーバーを起動**
   ```bash
   npm run dev
   ```

   ブラウザで `http://localhost:5173` を開くと、ゲームが起動します。

### ビルド

本番環境用のビルドを作成する場合：

```bash
npm run build
```

ビルド結果は `dist` ディレクトリに出力されます。

### プレビュー

ビルド結果をプレビューする場合：

```bash
npm run preview
```
