# アイコンファイルについて

このフォルダには、アプリのアイコンファイルを配置してください。

## 必要なファイル

以下のファイルを `public/` フォルダに配置してください：

1. **favicon.ico** (32x32 または 64x64)
   - ブラウザのタブに表示されるアイコン

2. **icon-192.png** (192x192)
   - PWA用のアイコン（中サイズ）

3. **icon-512.png** (512x512)
   - PWA用のアイコン（大サイズ）

## アイコンのデザイン

ノヴァリア王国の国章風シンボルとして、以下のデザインを推奨します：

- **配色**: 紺色（#1e293b）背景 × 金色（#fbbf24, #fcd34d）のシンボル
- **デザイン**: 星やダイヤモンドを組み合わせたシンボル
- **参考**: `icon.svg` にSVG版のアイコンがあります

## アイコンの生成方法

### 方法1: generate-icons.html を使用

1. `generate-icons.html` をブラウザで開く
2. 各サイズのアイコンが自動生成されます
3. 「ダウンロード」ボタンをクリックして保存

### 方法2: SVGから変換

1. `icon.svg` を画像編集ソフトで開く
2. 各サイズ（32x32, 192x192, 512x512）にリサイズ
3. PNG形式で保存

### 方法3: オンラインツールを使用

- [Favicon Generator](https://favicon.io/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)

これらのツールで、1つの画像から複数サイズのアイコンを生成できます。

## ファイル配置後の確認

アイコンファイルを配置したら、以下を確認してください：

1. `/favicon.ico` がブラウザのタブに表示されるか
2. `/manifest.json` が正しく読み込まれるか
3. `/icon-192.png` と `/icon-512.png` が存在するか
4. スマホで「ホーム画面に追加」が動作するか
